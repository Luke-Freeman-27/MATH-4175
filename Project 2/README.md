NAMES: Luke Freeman, Yoshwan Pathipati, Diego Penadillo, Hiren Sai Vellanki

PURPOSE: To first find the keyword length of the cipher. This is accomplished by calculating the index of coincidence. Next, the step is doing frequency analysis to find the possible keyword for the ciphertext. Finally, once the keyword is found, it is then decrypted into plaintext.

Type the command "python3 IC.py" in your given terminal in order to run this script. Our final answers are in the "results.md" file. The IC.py file also takes a "ciphertext.md" file for the provided ciphertext for this project. We can only submit PDF files so it may mean you need to copy and paste the ciphertext into a new ciphertext.md file for this to run correctly.
